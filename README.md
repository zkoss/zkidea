# zkidea
ZK IntelliJ IDEA Plugin

### IntelliJ Version

This plugin can support on IntelliJ platform 141.1532 or greater

### Features

 * ZUL content assist
 * ZK MVVM annotation content assist
 * ZUL syntax check
 * ZK Maven archetypes for creating ZK project
